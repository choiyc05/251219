### 2. 자료형
    
    [기본 타입] Primitive Type
		- number    : 숫자(정수, 실수)
		- bigint    : 정수 -(2의 53승 -1) ~ 2의 53승-1
		- string    : 문자열
		- boolean   : 논리값(true, false)
		- undefined : 초기화되지 않은 상태, 변수 초기화에 사용
		- null      : 객체 변수 초기화 사용
		- NaN       : Not a Number

		[참조 타입] 참조형, Reference Type
		- array
		- function
		- ...
