### 3. 문자열
	study03,04
	
	* 템플릿 리터럴 : 백틱 사용
    : 이중 따옴표나 작은 따옴표 대신 백틱(`)을 사용해서 표현
    : 작은 따옴표와 혼동할 수 있으나, 백틱은 일반적으로 키보드 tab 키 위에 있음
    : 플래이스 홀더를 이용한 ${expression}으로 표현식 사용

| 특징 | 일반 따옴표 (' ', " ") | 백틱 (` `) |
| :--- | :--- | :--- |
| **줄바꿈** | `\n` 사용 필수 | 엔터키로 바로 가능 |
| **변수 결합** | `+` 연산자 사용 (복잡함) | `${variable}` (간결함) |
| **가독성** | 낮음 (코드가 지저분해짐) | 높음 (한 문장으로 보임) |
